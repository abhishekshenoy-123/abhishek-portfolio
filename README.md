# R Abhishek Shenoy - Portfolio Website

A modern, responsive portfolio website built with React, featuring smooth animations and a professional design.

## 🚀 Features

- **Modern UI/UX**: Clean and professional design with Navy & Gold color scheme
- **Smooth Animations**: Powered by Framer Motion for engaging interactions
- **Responsive Design**: Fully responsive layout using Tailwind CSS
- **Fast Performance**: Built with Vite for optimal loading speeds
- **Interactive Components**: Animated cards, hover effects, and smooth scrolling
- **Contact Form**: Functional contact section with email integration
- **Social Links**: Quick access to GitHub, LinkedIn, and email

## 📋 Sections

- **Hero**: Introduction with call-to-action buttons
- **About**: Personal introduction and overview
- **Education**: Timeline of academic achievements
- **Skills**: Technical skills showcase with categories
- **Projects**: Portfolio of completed projects
- **Experience**: Work experience and extracurricular activities
- **Contact**: Contact form and information
- **Footer**: Quick links and social media

## 🛠️ Technologies Used

- **React 18**: Frontend framework
- **Vite**: Build tool and development server
- **Tailwind CSS**: Utility-first CSS framework
- **Framer Motion**: Animation library
- **React Icons**: Icon library

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/abhishekshenoy-123/abhishek-portfolio.git
```

2. Navigate to the project directory:
```bash
cd abhishek-portfolio
```

3. Install dependencies:
```bash
npm install
```

## 🏃 Running the Project

### Development Mode
```bash
npm run dev
```
The application will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
```
The optimized build will be in the `dist` directory

### Preview Production Build
```bash
npm run preview
```

## 🎨 Customization

### Color Scheme
Colors are defined in `tailwind.config.js`:
```javascript
colors: {
  primary: '#0c1929',    // Deep Navy
  secondary: '#1a3a52',  // Lighter Navy
  accent: '#d4af37',     // Gold
  accentLight: '#f59e0b', // Amber
  text: '#e5e7eb',       // Light Gray
}
```

### Content
All content is organized in the `src/components/` directory:
- `Hero.jsx` - Landing section
- `About.jsx` - About me section
- `Education.jsx` - Education timeline
- `Skills.jsx` - Skills showcase
- `Projects.jsx` - Project portfolio
- `Experience.jsx` - Work experience
- `Contact.jsx` - Contact information
- `Footer.jsx` - Footer with links

## 📤 Deployment

### Vercel (Recommended)
1. Push code to GitHub
2. Import repository in Vercel
3. Deploy automatically

### Netlify
1. Build the project: `npm run build`
2. Drag and drop the `dist` folder to Netlify

### GitHub Pages
1. Push code to GitHub
2. Enable GitHub Pages in repository settings
3. Select `main` branch as source
